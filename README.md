# VIBEBIOS 🖥️

**VIBEBIOS** — это кастомная 32-битная графическая прошивка (ROM BIOS), написанная с нуля на C и Assembly для архитектуры x86 (эмулятор QEMU). 

В отличие от стандартных загрузчиков (которые грузятся в `0x7C00`), этот проект является **настоящим BIOS**. Он компилируется в бинарный ROM-файл размером 128 КБ, мапится в самый конец 4 ГБ адресного пространства материнской платы и перехватывает управление прямо с вектора сброса процессора (`0xFFFFFFF0`).

## 🌟 Фичи
* **Мгновенный 32-битный режим:** Никакого долгого сидения в 16-битном реальном режиме. BIOS настраивает GDT и прыгает в Protected Mode в первые же микросекунды старта.
* **Без прерываний (No IVT):** Код общается с железом исключительно напрямую через порты ввода-вывода (I/O Ports). Никаких `int 10h` или `int 13h`.
* **High-Res Графика (800x600):** Работа с шиной PCI для настройки видеокарты и включения Linear Frame Buffer (LFB) в обход ограничения в 64 КБ классического VGA.
* **Собственный UI:** Интегрированный растровый шрифт и функции отрисовки примитивов.
* **ATA PIO Драйвер:** Аппаратное чтение нулевого сектора (MBR) жесткого диска.
* **Загрузка ОС:** Валидация сигнатуры `0x55 0xAA` и передача управления (JMP `0x7C00`) пользовательской операционной системе.

## 🛠️ Зависимости
Для сборки проекта под Linux потребуются:
* `gcc` (с поддержкой `-m32`)
* `nasm` (Ассемблер)
* `binutils` (для утилиты `ld` и `objcopy`)
* `qemu-system-i386` (Эмулятор)
* `make`

## 🚀 Быстрый старт
1. Склонируйте репозиторий.
2. В терминале выполните команду для компиляции и запуска:
   ```bash
   make run
   ```
3. Проект скомпилирует ядро BIOS (`bios.bin`), соберет тестовую ОС (`os.bin`) и запустит QEMU.
4. Для очистки директории от скомпилированных файлов используйте:
   ```bash
   make clean
   ```

## 📂 Структура проекта
* `boot.asm` — Стартовый ассемблерный код, Вектор Сброса процессора, GDT и переход в 32 бита.
* `bios.c` — Ядро BIOS на C (Инициализация PCI, VGA, драйвер ATA диска, отрисовка интерфейса).
* `linker.ld` — Скрипт линковщика, формирующий правильную архитектуру 128 KB ROM.
* `os.asm` — Минимальная тестовая ОС, демонстрирующая перехват управления у BIOS.
* `Makefile` — Скрипт автоматизации сборки.

## ⚠️ Дисклеймер
WILIXBIOS не является заменой SeaBIOS или UEFI. Он не предоставляет таблиц ACPI, e820 или обработчиков прерываний 16-битного режима. Стандартные загрузчики вроде GRUB или образы Ubuntu/Windows **не запустятся** на этом BIOS. Он предназначен для запуска кастомных 32-битных ОС, написанных специально под него.
так же репозиторий весь был создан искуственным интелектом.
